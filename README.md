# pv-training-v1

This is a simple game for training your peripheral vision

For the following steps for traing:
1. Focus on the red "Reload" ball
2. By using your peripheral vision, click the number in ascending order
3. If you click correctly, the number will change to green
4. If you click incorrecyly, at other number will change to red and you will need to click the red "Reload" ball to restart
5. Once you finish click the number in ascending order, click the red "Reload" ball to restart
